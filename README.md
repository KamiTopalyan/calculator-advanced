### The process

- In the project directory, run `npm start`
- Navigate to [http://localhost:3000](http://localhost:3000)
- Let the Page Load
- Input a Number
- Click an operation
- Input another Number
- Click equals
- Input another number
- Click an operation
- Click equals
- Repeat last 3 steps
